☁️ Supabase — Banco de Dados

Permite acessar seus lotes em qualquer dispositivo

Obtenha sua chave em console.anthropic.com → API Keys. A chave fica salva somente neste dispositivo.

Cole aqui a URL de implantação do seu Web App no Google Apps Script.

📌 Como configurar o backend Google Sheets:

1. Abra o Google Sheets e crie uma planilha nova.
2. Vá em Extensões → Apps Script e cole o código cacau_backend.gs.
3. Clique em Implantar → Novo deployment como Web App.
4. Em "Quem tem acesso" escolha Qualquer pessoa.
5. Copie a URL gerada e cole no campo acima.
